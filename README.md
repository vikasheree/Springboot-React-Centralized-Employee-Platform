# Springboot-React-Centralized-Employee-Platform
A full-stack Centralized Employee Workforce Platform built using Spring Boot, MySQL, and React.js.  The application supports CRUD operations for employees with a RESTful backend and a responsive frontend.

Features
- Add new employees
- View all employees
- Update employee details
- Delete employee records
- RESTful API architecture
- Responsive React UI
- MySQL database integration

Tech Stack - Backend
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL

Frontend
- React.js
- JavaScript
- HTML5
- CSS3

Tools
- Maven
- Git & GitHub
- VS Code
- MySQL Workbench
- Postman
